# Doom WASM

Browser-based build of **Chocolate Doom** using WebAssembly.

## Requirements

Place these files in the same folder:

- `index.html`
- `chocolate-doom.js`
- `chocolate-doom.wasm`
- `chocolate-doom.data`
- `doom1.wad`

## Run Locally

Use a local server:

```bash
python3 -m http.server 8000
```

## Then open:

http://localhost:8000

## Notes

- Do not open the HTML file directly.
- doom1.wad is required to play.
